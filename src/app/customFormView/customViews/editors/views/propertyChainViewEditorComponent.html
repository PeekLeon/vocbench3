With this view, the object in the ResourceView will be rendered with a single value reached passing through a defined property chain.
E.g.: $resource $trigprop ?o1 .
?o1 %prop1% ?o2 .
...
?oN %propN ?RENDERED_VALUE . Note: the property chain must start from the object of the triggering property, so such property doesn't have to be included in the chain.
